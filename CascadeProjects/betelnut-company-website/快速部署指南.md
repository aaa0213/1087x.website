## 📦 方法二：手动上传部署

### 1. 准备文件
- 网站文件位置：`C:\Users\DELL\CascadeProjects\betelnut-company-website`
- 压缩包：`1087x-website.tar.gz`

### 2. 上传到服务器
```bash
# 使用SCP上传
scp -r betelnut-company-website root@139.199.1.35:/tmp/

# 或上传压缩包
scp 1087x-website.tar.gz root@139.199.1.35:/tmp/
```

### 3. 服务器端操作
```bash
# 登录服务器
ssh root@139.199.1.35

# 如果上传了压缩包，先解压
cd /tmp
tar -xzf 1087x-website.tar.gz
mv betelnut-company-website /var/www/1087x

# 运行部署脚本
cd /var/www/1087x
chmod +x deploy.sh
./deploy.sh
```
